# ShortcutSoundboard for BetterDiscord

A BetterDiscord(BD) plugin that let you play soundboards sounds from keyboard shortcuts.

## Features:

- Bind global keybinds to play sounboards sounds by `shif+click` the sounboard button
- Set delay between sending sounds
- Enable/Disable Showing toast
