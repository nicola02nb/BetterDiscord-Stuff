# ShortcutScreenshareScreen for BetterDiscord

A BetterDiscord(BD) plugin that let you screenshare screen from keyboard shortcut when no game is running.

## Features:

- **Keybinds**:
  - Set shortcut to toggle Start/Stop screen streaming
  - Set shortcut to toggle Start/Stop device streaming
  - Set shortcut to toggle Start/Stop game streaming
  - Set shortcut to stop the current stream
  - Set shortcut to toggle switch between two sources
  - Set shortcut to toggle audio sharing
  - Set shortcut to toggle preview
- **Default Sources**:
  - Set the default display to stream
  - Set the default device to stream
- **Stream Options**:
  - Enable/Disable stream preview
  - Enable/Disable audio sharing
- Set keybind behavior when already streaming (Stop Stream / Toggle Source)
- Enable/Disable showing toasts when stream starts or stops
